# 🍕 Food Delivery App

A complete food delivery application built with React Native and Node.js.

## 📁 Project Structure

```
food-delivery-app/
├── mobile/              # React Native mobile app
├── backend/             # Node.js Express API
├── database/            # Database schemas
└── docs/                # Documentation
```

## 🚀 Features

### Delivery Partner Features
- ✅ Online/Offline status management
- 📍 Real-time location tracking
- 📦 Accept/Reject orders
- 💰 Earnings tracking
- ⭐ Rating system

### User Features
- 🔍 Search restaurants
- 🛒 Shopping cart
- 💳 Payment integration
- 📦 Order tracking
- 💬 Chat support

## 🛠️ Tech Stack

- **Frontend**: React Native
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Real-time**: Socket.io
- **Location**: Google Maps API
- **Payments**: Stripe/Razorpay
